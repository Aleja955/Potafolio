Sobre mí

Soy Aleja, una desarrolladora Fullstack Java con experiencia en JavaScript, HTML, CSS y bases de datos. Me especializo en crear aplicaciones web modernas, funcionales y con un diseño atractivo. Mi formación en Artes Visuales me permite aportar una perspectiva estética única a mis proyectos, enfocándome tanto en la tecnología como en la experiencia del usuario.

Me apasiona el aprendizaje constante y la innovación, siempre buscando nuevos desafíos para seguir creciendo como profesional. Estoy emocionada por la oportunidad de aportar mis habilidades en desarrollo y diseño a proyectos que hagan la diferencia.

Proyectos

Eco-store

Este es un sitio web informativo para una tienda de productos artesanales y ecológicos. El proyecto está diseñado para ser completamente responsive, accesible y visualmente agradable, utilizando tecnologías como HTML, CSS y Saas. La tienda está orientada a promover productos sostenibles y ofrecer una experiencia de compra amigable con el medio ambiente.

Batatabit

Este proyecto fue creado como parte de mi práctica en HTML y CSS. Es una réplica de un sitio web real con el objetivo de practicar la maquetación web, asegurando que sea completamente responsive y tenga una estructura clara y visualmente atractiva. A través de este proyecto, perfeccioné mis habilidades en diseño web y optimización para dispositivos móviles.

Ver repositorio

Google Clone

Recreé la página de inicio de Google como una práctica de maquetación web. El proyecto se centró en la estructura responsive y los detalles de la interfaz, con un diseño lo más cercano posible al original, utilizando solo HTML y CSS. Fue una excelente forma de entender cómo funcionan los elementos de la interfaz y cómo optimizar su disposición en diferentes tamaños de pantalla.
